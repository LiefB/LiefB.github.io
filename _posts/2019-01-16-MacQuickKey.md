# MacQuickKey

```
command+C            复制
command+V            粘贴
command+X            剪切
command+Z            撤销
command+A            全选
del                  前删
fn+del               后删
command+←/→          home/end
command+control+F    全屏
control+←/→          应用间左右切换
command+tab          应用间切换
Fn11                 显示桌面
control+1            回到桌面
control+option+T     在当前目录打开终端
command+Q            关闭当前应用
command+option+esc   任务管理器
option+space         欧陆词典
shift+command+3/4    全屏/区域快照
```

# Gesture

```
二指                  页面滚动
三指                  三指拖动
四指                  调度中心
```

# Touchbar
一级：屏幕亮度、声音控制、回到桌面
二级：屏幕快照、锁定屏幕、键盘亮度
